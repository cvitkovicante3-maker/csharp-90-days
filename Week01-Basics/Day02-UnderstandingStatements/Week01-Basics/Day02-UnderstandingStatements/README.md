# Day 2: Understanding Statements

## 📚 Concept
Code runs line-by-line, top to bottom, like a recipe. Each line ending in `;` is a statement.

## 💻 My Code
```csharp
using System;

class Program
{
    static void Main()
    {
        Console.WriteLine("Loading game...");
        Console.WriteLine("Loading complete!");
        Console.WriteLine("Starting engine...");
    }
}
