# Day 3: Variables - Storage Boxes

## 📚 Concept
Variables are labeled boxes that store data while your program runs. 
You declare the type, give it a name, then assign a value.

## 💻 My Code
```csharp
using System;




class Program
{
    static void Main()
    {
        string playerName;
        playerName = "Alex";
        Console.WriteLine(playerName);
    }





}
